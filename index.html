¡Bienvenido a mi sitio web!

Este sitio fue creado como parte del Curso de HTML para Principiantes. Aquí aprenderás los conceptos básicos del lenguaje de marcado que da vida a todas las páginas web del mundo.

HTML significa HyperText Markup Language y es la estructura fundamental de cualquier página web.

📚 ¿Qué aprendemos en este curso?

  • Estructura básica de un documento HTML
  • Etiquetas de texto: párrafos, títulos, énfasis
  • Listas ordenadas y no ordenadas
  • Imágenes y enlaces (hipervínculos)
  • Tablas para organizar información
  • Formularios para recibir datos del usuario

🏷️ Etiquetas HTML más usadas

Etiqueta Descripción Ejemplo
<h1> – <h6>Títulos o encabezadosTítulo principal
<p>Párrafo de textoEste es un párrafo
<a>Enlace o hipervínculoHaz clic aquí
<img>ImagenMuestra una imagen
<ul> / <ol>ListasLista de elementos
<table>Tabla de datosFilas y columnas
<form>FormularioDatos de usuario
<div>Contenedor genéricoSección de diseño

🔢 Pasos para crear una página web

  1. Abre un editor de texto (VS Code, Notepad++, etc.)
  2. Escribe la estructura básica HTML
  3. Agrega el contenido con las etiquetas adecuadas
  4. Guarda el archivo con la extensión .html
  5. Abre el archivo en tu navegador favorito
  6. ¡Listo! Tu primera página web está lista
