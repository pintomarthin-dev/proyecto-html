# 🌐 Proyecto HTML para Principiantes

Proyecto creado como parte del curso [HTML para Principiantes](https://youtu.be/rbuYtrNUxg4).

## 📄 Archivos

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Página principal con tabla de etiquetas y contenido |
| `about.html` | Página "Sobre Mí" con perfil y habilidades |
| `contacto.html` | Página de contacto con formulario HTML |
| `evidencias.pdf` | PDF con capturas de pantalla del proceso |
| `evidencias.docx` | Documento Word con evidencias del proceso |

## 🛠️ Conceptos aplicados

- Estructura HTML5 (`DOCTYPE`, `html`, `head`, `body`)
- Etiquetas semánticas (`header`, `nav`, `main`, `footer`)
- Títulos, párrafos, listas y tablas
- Formularios con `input`, `select`, `textarea`
- Estilos CSS básicos inline
- Navegación entre páginas con hipervínculos
